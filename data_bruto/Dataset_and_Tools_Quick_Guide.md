# Dataset and Tools Quick Guide

## Core datasets to use first

1. Angola sample tourism attractions: `data/sample/angola_tourism_attractions_sample.csv`
2. HOTOSM Angola POIs: `https://data.humdata.org/dataset/hotosm_ago_points_of_interest`
3. Angola administrative boundaries: `https://data.humdata.org/dataset/cod-ab-ago`
4. HOTOSM Angola roads: `https://data.humdata.org/dataset/hotosm_ago_roads`
5. NASA POWER climate API/viewer: `https://power.larc.nasa.gov/data-access-viewer/`
6. World Bank Angola indicators: `https://data.worldbank.org/country/angola`
7. Geofabrik Angola OpenStreetMap extract: `https://download.geofabrik.de/africa/angola.html`
